## 📦 Installation
1. Clone the repository.
```sh
$ git clone https://github.com/turkerssh/discord.js-v14-bot-template.git
```
2. Install the dependencies.
```sh
$ npm install
```
3. Rename the `exemple.config.js` file to `config.js` and fill in the required fields.
4. Start the bot.
```sh
$ npm run start
```
